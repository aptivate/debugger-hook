== Description ==

This plugin provides a function called `debugger()`, which can be used by
eg geben-on-emacs.

[Follow this project on Github](https://github.com/aptivate/debugger-hook)


== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory.
2. Activate it through the **Plugins** menu in WordPress.
